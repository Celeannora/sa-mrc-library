# Category 12 — Removable Media Controls


## JSIG Applicability
This category maps to JSIG-required controls. All tasks require written ISSM authorization (MA-2) and are retained as Body of Evidence (BoE) artifacts for the ATO package and ConMon program.

## Applicable JSIG / NIST SP 800-53 Controls
| Control | Title | JSIG Requirement |
|---------|-------|-----------------|
| MP-7 | Media Use | Required — restrict or prohibit removable media |
| MP-2 | Media Access | Required — restrict access to media |
| MP-5 | Media Transport | Required — control transport of media outside SAPF |
| CM-7 | Least Functionality | Required — disable unused ports/functions |

## JSIG-Specific Notes
- **PSO authorizes ALL digital media and media use in a SAP environment (DoDM 5205.07).**
- Any unauthorized removable media event is a potential spillage incident — treat as security incident.
- Media sanitization procedures must be approved by PSO per DoDM 5205.07.
- USB/removable media used during maintenance must have PSO written approval before entering SAPF.

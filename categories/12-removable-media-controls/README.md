# Category 12 — Removable Media Controls

## Description
Enforcement verification of USB block/allowlist policies, removable media inventory audit, and authorization record review.

## Applicable NIST Controls
- MP-7 — Media Use
- MP-2 — Media Access
- CM-7 — Least Functionality

## MRCs in This Category

| MRC ID | Task Title | Periodicity |
|--------|-----------|-------------|
| MRC-1201-MO | Monthly Removable Media Policy Enforcement Verification | Monthly |
| MRC-1202-QR | Quarterly Removable Media Inventory and Authorization Audit | Quarterly |

## Notes
- Unauthorized removable media use is a potential data exfiltration vector and must be treated as a security incident.

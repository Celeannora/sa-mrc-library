# Category 19 — Documentation & Records Management


## JSIG Applicability
This category maps to JSIG-required controls. All tasks require written ISSM authorization (MA-2) and are retained as Body of Evidence (BoE) artifacts for the ATO package and ConMon program.

## Applicable JSIG / NIST SP 800-53 Controls
| Control | Title | JSIG Requirement |
|---------|-------|-----------------|
| PL-2 | System Security Plan | Required — maintain current accurate SSP |
| AU-11 | Audit Record Retention | Required — retain audit records per policy |
| SA-5 | System Documentation | Required — maintain operator and user documentation |
| PM-4 | Plan of Action and Milestones Process | Required — manage and track POA&M |
| CA-7 | Continuous Monitoring | All MRCs are documentation artifacts for ConMon |

## JSIG-Specific Notes
- ISSM maintains a cybersecurity documentation repository including all ATOs (DoDM 5205.07).
- SSP must be reviewed annually and updated whenever the system changes.
- All signed MRCs must be retained per AU-11 and applicable federal records management policy.
- POA&M must be current and submitted to ISSM monthly — delays may affect ATO status.

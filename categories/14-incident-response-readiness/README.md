# Category 14 — Incident Response Readiness

## Description
IR playbook currency review, tabletop exercise coordination, escalation path validation, and incident ticketing system health checks.

## Applicable NIST Controls
- IR-3 — Incident Response Testing
- IR-4 — Incident Handling
- IR-8 — Incident Response Plan

## MRCs in This Category

| MRC ID | Task Title | Periodicity |
|--------|-----------|-------------|
| MRC-1401-QR | Quarterly Incident Response Plan and Escalation Path Review | Quarterly |
| MRC-1402-AN | Annual IR Tabletop Exercise | Annual |

## Notes
- IR plan must be reviewed after any actual incident — not just on schedule.

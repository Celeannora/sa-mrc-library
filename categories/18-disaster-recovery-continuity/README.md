# Category 18 — Disaster Recovery / Continuity


## JSIG Applicability
This category maps to JSIG-required controls. All tasks require written ISSM authorization (MA-2) and are retained as Body of Evidence (BoE) artifacts for the ATO package and ConMon program.

## Applicable JSIG / NIST SP 800-53 Controls
| Control | Title | JSIG Requirement |
|---------|-------|-----------------|
| CP-2 | Contingency Plan | Required — develop, maintain, approve contingency plan |
| CP-4 | Contingency Plan Testing | Required — test plan periodically |
| CP-6 | Alternate Storage Site | Required — identify and maintain alternate storage |
| CP-7 | Alternate Processing Site | Required — identify and maintain alternate processing |
| CP-9 | System Backup | Required — backup and test restoration |

## JSIG-Specific Notes
- DR exercises are BoE artifacts for CP-4 and CA-7 ConMon.
- Alternate sites must maintain the same SAP security controls as the primary SAPF.
- ISSM must review and approve the contingency plan (CP-2).
- All media used in DR/COOP activities must be authorized by PSO.

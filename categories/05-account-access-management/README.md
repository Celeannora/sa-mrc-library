# Category 05 — Account & Access Management

## Description
User account provisioning/de-provisioning review, privileged account recertification, dormant account identification, group membership audit, and least-privilege validation.

## Applicable NIST Controls
- AC-2 — Account Management
- AC-3 — Access Enforcement
- AC-6 — Least Privilege
- IA-4 — Identifier Management
- IA-5 — Authenticator Management

## MRCs in This Category

| MRC ID | Task Title | Periodicity |
|--------|-----------|-------------|
| MRC-0501-WK | Weekly Dormant and Anomalous Account Review | Weekly |
| MRC-0502-MO | Monthly Privileged Account Recertification | Monthly |
| MRC-0503-QR | Quarterly Full Access Control Audit | Quarterly |

- Dormant accounts must be disabled after [X] days per policy.

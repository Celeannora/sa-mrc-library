## Procedure Steps

| Step | Action | Expected Result |
|------|--------|----------------|
| 1 | Export current privileged account list from Active Directory / PAM tool. | List exported. |
| 2 | Cross-reference each account against the current authorized user roster. | All accounts have a valid authorization on file. |
| 3 | Identify any accounts with no recent activity (>30 days). Document. | No unexplained dormant privileged accounts. |
| 4 | Identify any accounts assigned to departed or transferred personnel. Disable immediately. | No unauthorized accounts active. |
| 5 | Validate group memberships for each privileged account. Confirm least privilege. | Memberships consistent with role. |
| 6 | Document all changes made and submit to ISSM for awareness. | Change log complete. |
| 7 | Sign and date MRC. | MRC complete. |

# Category 07 — Network & Boundary Security

## Description
Firewall rule review and ACL validation, VPN utilization and connectivity, IDS/IPS sensor and alert queue review, DMZ and enclave boundary checks.

## Applicable NIST Controls
- SC-7 — Boundary Protection
- SC-5 — Denial of Service Protection
- SI-4 — System Monitoring
- CA-3 — Information Exchange

## MRCs in This Category
| MRC ID | Task Title | Periodicity |
|--------|-----------|-------------|
| MRC-0701-DA | Daily IDS/IPS and Network Alert Review | Daily |
| MRC-0702-QR | Quarterly Firewall Rule and ACL Audit | Quarterly |

## Notes
Firewall rule changes require CCB approval and ISSM notification. No temporary rules without written authorization.

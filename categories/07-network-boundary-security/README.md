# Category 07 — Network & Boundary Security


## JSIG Applicability
This category maps to JSIG-required controls. All tasks require written ISSM authorization (MA-2) and are retained as Body of Evidence (BoE) artifacts for the ATO package and ConMon program.

## Applicable JSIG / NIST SP 800-53 Controls
| Control | Title | JSIG Requirement |
|---------|-------|-----------------|
| SC-7 | Boundary Protection | Required — monitor and control external boundary communications |
| CA-3 | Information Exchange | Required — ISAs required for all interconnections |
| SI-4 | System Monitoring | Required — network monitoring for attacks |
| AC-4 | Information Flow Enforcement | Required — enforce approved information flow policies |
| MA-4 | Nonlocal Maintenance | Required — remote sessions must use approved encrypted authenticated channels |

## JSIG-Specific Notes
- No firewall rule changes without CCB-approved CR and ISSM authorization (CM-3).
- All network interconnections require an approved Interconnection Security Agreement (ISA) per CA-3.
- Nonlocal (remote) maintenance requires ISSM approval and must be logged (MA-4).
- MA-4 remote sessions must be conducted over authenticated, encrypted channels only.

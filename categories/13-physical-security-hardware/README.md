# Category 13 — Physical Security & Hardware

## Description
Server room and data center access log review, equipment seal inspection, hardware lifecycle and EOL assessment, UPS/generator testing, and firmware/BIOS currency.

## Applicable NIST Controls
- PE-2 — Physical Access Authorizations
- PE-3 — Physical Access Control
- PE-6 — Monitoring Physical Access
- MA-2 — Controlled Maintenance

## MRCs in This Category

| MRC ID | Task Title | Periodicity |
|--------|-----------|-------------|
| MRC-1301-DA | Daily Physical Access Log and Equipment Visual Check | Daily |
| MRC-1302-QR | Quarterly Hardware Lifecycle and Firmware Review | Quarterly |
| MRC-1303-SA | Semi-Annual UPS and Generator Test | Semi-Annual |

## Notes
- Physical access anomalies must be reported to the FSO and ISSM same day.

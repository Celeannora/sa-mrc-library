# Category 13 — Physical Security & Hardware


## JSIG Applicability
This category maps to JSIG-required controls. All tasks require written ISSM authorization (MA-2) and are retained as Body of Evidence (BoE) artifacts for the ATO package and ConMon program.

## Applicable JSIG / NIST SP 800-53 Controls
| Control | Title | JSIG Requirement |
|---------|-------|-----------------|
| PE-2 | Physical Access Authorizations | Required — maintain authorized individual lists |
| PE-3 | Physical Access Control | Required — enforce physical access controls at SAPF |
| PE-6 | Monitoring Physical Access | Required — monitor physical access to IS |
| MA-2 | Controlled Maintenance | Physical maintenance is controlled maintenance |
| MA-5 | Maintenance Personnel | Required — cleared personnel only; escort if not cleared |

## JSIG-Specific Notes
- All physical access to SAPF and IS must be by cleared, authorized personnel (MA-5, PE-2).
- Personnel without required clearance/SAP access must be ESCORTED by a cleared SA (MA-5).
- PSO provides written approval for entry/removal of IS from the SAPF (DoDM 5205.07).
- GSSO/CSSO conducts and documents **annual SAPF self-inspections** (DoDM 5205.07).
- TEMPEST requirements are issued by the PSO — maintenance must not compromise TEMPEST posture.

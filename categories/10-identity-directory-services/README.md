# Category 10 — Identity & Directory Services

## Description
Active Directory replication health, DNS zone integrity, DHCP scope review, Group Policy validation, and domain controller health checks.

## Applicable NIST Controls
- IA-2 — Identification and Authentication (Organizational Users)
- IA-4 — Identifier Management
- CM-6 — Configuration Settings
- SC-20 — Secure Name / Address Resolution Service

## MRCs in This Category
| MRC ID | Task Title | Periodicity |
|--------|-----------|-------------|
| MRC-1001-DA | Daily Domain Controller Replication and Health Check | Daily |
| MRC-1002-MO | Monthly DNS Zone Integrity and DHCP Scope Review | Monthly |
| MRC-1003-QR | Quarterly Group Policy Audit | Quarterly |

## Notes
AD replication failures must be escalated to ISSO within 4 hours. Domain Controller health directly impacts authentication and audit log integrity.

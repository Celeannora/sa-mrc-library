# Category 06 — System Health & Performance

## Description
CPU, memory, disk, and network utilization monitoring; service availability checks; hardware alert review (iLO/iDRAC); RAID and storage health.

## Applicable NIST Controls
- CA-7 — Continuous Monitoring
- SI-4 — System Monitoring
- CP-9 — System Backup (storage)

## MRCs in This Category
| MRC ID | Task Title | Periodicity |
|--------|-----------|-------------|
| MRC-0601-DA | Daily System Health Dashboard Review | Daily |
| MRC-0602-MO | Monthly Resource Utilization Trend Analysis | Monthly |

## Notes
Persistent resource anomalies must be documented and a remediation plan submitted to ISSM.

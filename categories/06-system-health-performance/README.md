# Category 06 — System Health & Performance


## JSIG Applicability
This category maps to JSIG-required controls. All tasks require written ISSM authorization (MA-2) and are retained as Body of Evidence (BoE) artifacts for the ATO package and ConMon program.

## Applicable JSIG / NIST SP 800-53 Controls
| Control | Title | JSIG Requirement |
|---------|-------|-----------------|
| CA-7 | Continuous Monitoring | Required — monitor security controls and system status |
| SI-4 | System Monitoring | Required — active monitoring for attacks and indicators |
| MA-2 | Controlled Maintenance | Health check records are MA-2 artifacts |
| MA-6 | Timely Maintenance | Required — prevent failures that affect availability |

## JSIG-Specific Notes
- Persistent resource anomalies must be documented and a remediation plan submitted to ISSM.
- SI-4 requires the SA to detect and report unauthorized use — health anomalies may indicate compromise.
- ISSM ensures SAs monitor vulnerabilities and attacks (DoDM 5205.07 ISSM responsibilities).

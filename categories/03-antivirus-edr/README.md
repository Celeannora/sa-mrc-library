# Category 03 — Antivirus / Endpoint Detection & Response (EDR)

## Description
Definition currency verification, scan engine integrity, agent health across all managed endpoints, quarantine review, and EDR policy compliance.

## Applicable NIST Controls
- SI-3 — Malicious Code Protection
- SI-8 — Spam Protection
- CM-6 — Configuration Settings

## MRCs in This Category

| MRC ID | Task Title | Periodicity |
|--------|-----------|-------------|
| MRC-0301-DA | Daily AV/EDR Definition and Agent Health Verification | Daily |
| MRC-0302-MO | Monthly EDR Policy Compliance and Full Scan Review | Monthly |

## Notes
- Any offline or non-compliant agent must be remediated same-day or escalated.
- Quarantine items must be reviewed and disposition documented.

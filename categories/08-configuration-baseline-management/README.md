# Category 08 — Configuration & Baseline Management

## Description
Configuration drift detection against approved baseline, STIG/SCAP scan execution, software inventory reconciliation against the Approved Software List (ASL), and CMDB updates.

## Applicable NIST Controls
- CM-2 — Baseline Configuration
- CM-6 — Configuration Settings
- CM-7 — Least Functionality
- CM-8 — System Component Inventory
- SA-10 — Developer Configuration Management

## MRCs in This Category
| MRC ID | Task Title | Periodicity |
|--------|-----------|-------------|
| MRC-0801-MO | Monthly STIG/SCAP Scan and Drift Detection | Monthly |
| MRC-0802-MO | Monthly Software Inventory Reconciliation | Monthly |
| MRC-0803-QR | Quarterly Baseline Configuration Review | Quarterly |

## Notes
All detected drift must be documented and remediated or accepted via formal risk acceptance signed by the AO.
